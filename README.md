# Sudoku Solver in C++

## Overview
This project implements a 9x9 Sudoku solver in C++ using backtracking and bitset optimization.  
It reads unsolved Sudoku boards from terminal input and prints the solved board.

## Features
- Reads Sudoku board from terminal (0 for empty cells).
- Solves using recursive backtracking.
- Optimized with bitsets to prune search space.
- Prints solved board neatly in terminal.
